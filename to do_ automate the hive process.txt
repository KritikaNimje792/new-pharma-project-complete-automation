## ✅ Option 1: Automate using a Shell Script (Best Practice)

### 1️⃣ Create a Sqoop import script

```bash
vi /home/hadoop/sqoop_qc_results_import.sh
```

Paste this:

```bash
#!/bin/bash

# -------------------------------
# Variables
# -------------------------------
DB_HOST="retailpulse-kritika-db.c58qessmggt4.ap-south-1.rds.amazonaws.com"
DB_PORT="3306"
DB_NAME="pharma_kritika"
TABLE_NAME="qc_results"
HDFS_TARGET_DIR="/user/hadoop/qc_results"
MYSQL_USER="admin"
PASSWORD_FILE="/home/hadoop/.mysql_pass"
LOG_DIR="/home/hadoop/sqoop_logs"
TIME_STAMP=$(date '+%Y-%m-%d_%H-%M-%S')
LOG_FILE="$LOG_DIR/qc_results_import_$TIME_STAMP.log"

# -------------------------------
# Create log directory if missing
# -------------------------------
mkdir -p $LOG_DIR

echo "Starting Sqoop Import at $(date)" >> $LOG_FILE

# -------------------------------
# Run Sqoop
# -------------------------------
sqoop import \
--connect jdbc:mysql://${DB_HOST}:${DB_PORT}/${DB_NAME} \
--username ${MYSQL_USER} \
--password-file ${PASSWORD_FILE} \
--driver com.mysql.cj.jdbc.Driver \
--table ${TABLE_NAME} \
--target-dir ${HDFS_TARGET_DIR} \
--as-textfile \
--fields-terminated-by ',' \
--lines-terminated-by '\n' \
--m 1 >> $LOG_FILE 2>&1

# -------------------------------
# Status check
# -------------------------------
if [ $? -eq 0 ]; then
  echo "Sqoop Import SUCCESS at $(date)" >> $LOG_FILE
else
  echo "Sqoop Import FAILED at $(date)" >> $LOG_FILE
fi
```

--connect jdbc:mysql://... \
--username admin \
